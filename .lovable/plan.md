

## خطة: نسبة خصم لكل منطقة مع حساب تلقائي للسعر

### الفكرة
بدلاً من إدخال سعر كل منطقة يدوياً، يُدخل الأدمن **السعر الافتراضي** + **نسبة خصم لكل منطقة** (%)، ويُحسب سعر المنطقة تلقائياً:
`سعر المنطقة = السعر الافتراضي × (1 - نسبة الخصم / 100)`

### التغييرات

**1. Migration — إضافة عمودي الخصم**
- إضافة `discount_zone_a numeric default 0` و `discount_zone_b numeric default 0` لجدول `subscription_plans`

**2. تعديل `AdminSubscriptionPlans.tsx`**
- استبدال حقول إدخال أسعار المناطق بحقلي **نسبة خصم** (%) لكل منطقة
- حساب `price_zone_a` و `price_zone_b` تلقائياً عند تغيير السعر الافتراضي أو نسبة الخصم
- عرض السعر المحسوب بجانب حقل الخصم (للمعاينة الفورية)
- تحديث `handleSave` لحفظ نسب الخصم + الأسعار المحسوبة معاً
- تحديث الجدول ليعرض نسبة الخصم بجانب سعر كل منطقة

**3. تعديل واجهة التعديل (Dialog)**
- تخطيط جديد:
  - صف 1: السعر الافتراضي
  - صف 2: منطقة أ → نسبة الخصم % → السعر المحسوب (للقراءة فقط)
  - صف 3: منطقة ب → نسبة الخصم % → السعر المحسوب (للقراءة فقط)
