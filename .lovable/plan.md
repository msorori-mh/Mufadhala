

## تصدير دليل القبول كـ PDF

### التغييرات — ملف واحد: `src/pages/CollegeGuide.tsx`

1. **إضافة زر "تصدير PDF"** بجانب الفلاتر الحالية
2. **إنشاء دالة `exportGuideAsPDF`** تستخدم نفس أسلوب `exportToPDF` الموجود (فتح نافذة طباعة HTML مع RTL) — لكن بتصميم مخصص لدليل القبول يعرض:
   - عنوان "دليل الكليات والمتطلبات" مع التاريخ
   - جدول يتضمن: اسم الكلية، الجامعة، الحد الأدنى للمعدل، نسبة القبول، موعد التنسيق، الوثائق المطلوبة، ملاحظات
   - تجميع الكليات حسب الجامعة لسهولة القراءة
3. **يُصدّر البيانات المفلترة حالياً** (حسب الجامعة أو البحث النصي المُطبّق)
4. استيراد أيقونة `Download` من lucide-react للزر

### النتيجة
زر تصدير يظهر بجانب الفلاتر، ينتج ملف PDF منسق بالعربية يحتوي على جميع بيانات القبول المفلترة مجمّعة حسب الجامعة.
