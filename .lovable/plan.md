
## ميزة: استيراد نموذج اختبارات سابق من ملف Excel

### الموقع
`src/pages/admin/AdminPastExams.tsx` — إضافة زر "استيراد من ملف" داخل محرر أسئلة النموذج (`QuestionsEditor`).

### تنسيق الملف المتوقع
ملف Excel/CSV بصف واحد للعناوين ثم صفوف الأسئلة. أعمدة (نفس نمط `importEngine`):

| العمود | مطلوب | ملاحظات |
|---|---|---|
| نص السؤال | ✅ | |
| الخيار أ | ✅ | |
| الخيار ب | ✅ | |
| الخيار ج | ❌ | فارغ مسموح |
| الخيار د | ❌ | فارغ مسموح |
| الإجابة الصحيحة | ✅ | a/b/c/d أو أ/ب/ج/د |
| الشرح | ❌ | |
| ترتيب العرض | ❌ | افتراضي: تسلسلي |

### الخطوات

**1. إنشاء `src/services/pastExamImport.ts`** — محلِّل خفيف خاص بأسئلة النموذج:
- `parsePastExamFile(data, fileName)` → يُرجع `{ questions, errors }`
- يستخدم نفس مكتبة `xlsx` الموجودة
- يطبّق نفس قواعد التحقق (إجبارية النص + الخيارَين أ/ب + الإجابة، تطبيع a-d/أ-د)
- يكتشف الأسئلة المكررة داخل الملف (تحذير فقط)

**2. تحديث `QuestionsEditor` في `AdminPastExams.tsx`**:
- زر "استيراد من ملف" بجانب زر "رجوع"
- زر "تحميل قالب فارغ" (يُولّد ملف Excel نموذجي محلياً عبر `xlsx`)
- `<input type="file" accept=".xlsx,.xls,.csv" hidden>` مع `useRef`
- بعد القراءة → عرض **معاينة** (Dialog) فيها:
  - عدد الأسئلة الصالحة + عدد الأخطاء
  - جدول مبسّط بأول 10 أسئلة
  - أزرار "إلغاء" / "استيراد N سؤال"
- عند التأكيد: `supabase.from("past_exam_model_questions").insert([...])` دفعة واحدة
- `order_index` يبدأ من `max(existing) + 1` للحفاظ على الأسئلة الحالية
- `refetch()` بعد النجاح مع toast بالنتيجة

**3. تهيئة الفورم (القالب)**:
- زر "تحميل قالب Excel" → ينشئ ملف `.xlsx` به ورقة واحدة "الأسئلة" بالعناوين العربية + سطر مثال واحد كنموذج توضيحي

### ضمانات الجودة
- ✅ لن يُمَس `importEngine.ts` (محرك الدروس) — معزول تماماً
- ✅ لن يُمَس جدول `past_exam_models` أو هيكلية النموذج
- ✅ التحقق client-side قبل أي `insert` لقاعدة البيانات
- ✅ الإدراج دفعة واحدة (atomic per-batch) لتجنب صفوف يتيمة
- ✅ يحترم RLS الموجودة (`Staff can manage model questions`)
- ✅ معالجة أخطاء واضحة: عدد السطر + سبب الرفض

### ما لن يتضمنه (لتجنب توسيع النطاق)
- ❌ لا استيراد لبيانات النموذج نفسه (العنوان/الجامعة/السنة) — يبقى يدوياً
- ❌ لا تعديل `importEngine.ts` (له منطق دروس مختلف)
- ❌ لا حذف للأسئلة الموجودة — الاستيراد **يُضيف فقط**
