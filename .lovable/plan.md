

# ربط مواد الاختبار بالكليات عبر جدول college_subjects

## ملخص
استخراج بيانات المواد المقررة لكل كلية من الملف المرفق وإدراجها في جدول `college_subjects` الموجود. يتطلب أولاً إنشاء المواد الدراسية الناقصة ثم ربطها بالكليات.

## المواد المطلوبة

**موجودة:** الكيمياء، الأحياء
**مطلوب إنشاؤها (5 مواد جديدة):**

| المادة | الكود |
|--------|-------|
| اللغة الإنجليزية | english |
| الرياضيات | math |
| الفيزياء | physics |
| الحاسوب | computer |
| اللغة العربية | arabic |

## خريطة الربط (من الملف المرفق)

**نمط طبي** (كيمياء + أحياء + إنجليزي) — 15 كلية:
سبأ-الطب، تعز-الطب، تعز-الطبية بالتربة، عدن-الطب، المهرة-طب أسنان، سيئون-الطب، حضرموت-الطب، ذمار-الطب البشري، ذمار-طب الأسنان، ذمار-العلوم الطبية، إب-الطب، إب-طب الأسنان، صنعاء-الطب، صنعاء-طب الأسنان، صنعاء-العلوم الطبية التطبيقية

**نمط هندسي/حاسوب** (رياضيات + فيزياء + إنجليزي + حاسوب) — 14 كلية:
سبأ-تكنولوجيا المعلومات، تعز-السعيد للهندسة، تعز-العلوم التطبيقية، تعز-حاسبات بالتربة، عدن-الهندسة، عدن-الحاسوب، شبوة-الحاسوب، المهرة-تقنية المعلومات، حضرموت-الحاسبات، حضرموت-هندسة حاسوب، ذمار-الهندسة، ذمار-الحاسبات، إب-الهندسة، إب-علوم الحاسوب، صنعاء-الهندسة، صنعاء-الحاسوب

**نمط إداري** (رياضيات + إنجليزي) — 5 كليات:
تعز-العلوم الإدارية، تعز-الإدارة بالتربة، حضرموت-إدارة أعمال، إب-المحاسبة، صنعاء-التجارة والاقتصاد

## خطوات التنفيذ

### 1. إنشاء المواد الجديدة (migration)
إضافة 5 مواد دراسية جديدة في جدول `subjects` بقيم UUID ثابتة.

### 2. إدراج بيانات الربط (insert tool)
إدراج ~120 سجل في `college_subjects` لربط كل كلية بموادها المقررة.

### 3. لا تغييرات في الكود
واجهة إدارة المواد تدعم عرض وإدارة `college_subjects` بالفعل.

## التفاصيل التقنية
- إنشاء المواد عبر migration لأنها تغيير في البنية (بيانات مرجعية أساسية)
- إدراج الروابط عبر insert tool (بيانات فقط)
- إجمالي السجلات: ~120 رابط (37 كلية × متوسط 3 مواد)
