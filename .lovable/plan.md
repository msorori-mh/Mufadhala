

## إعادة تصميم لوحة التحكم — تصغير وترتيب الخيارات

### المشكلة
بطاقات التنقل (navCards) كبيرة الحجم وتأخذ مساحة واسعة، مما يجعل المستخدم يحتاج للتمرير كثيراً قبل رؤية جميع الخيارات واختيار أحدها.

### الحل
تحويل بطاقات التنقل من تصميم بطاقات كبيرة (Card مع Header + Content) إلى تصميم مضغوط بشبكة 3 أعمدة على الجوال و4 أعمدة على الشاشات الأكبر، حيث يظهر كل خيار كأيقونة مع عنوان قصير فقط بدون وصف.

### التغييرات — ملف واحد: `src/pages/Dashboard.tsx`

**1. تعديل قسم التنقل (السطور 342-362)**
- استبدال شبكة `sm:grid-cols-2` بشبكة `grid-cols-3 sm:grid-cols-4`
- تحويل كل بطاقة إلى عنصر مضغوط: أيقونة دائرية + عنوان قصير تحتها (بدون وصف)
- إظهار شارة الإشعارات كنقطة صغيرة فوق الأيقونة
- حجم كل عنصر ~80px ارتفاع تقريباً بدلاً من ~120px

**2. شكل العنصر الجديد:**
```text
┌──────────┐  ┌──────────┐  ┌──────────┐
│   (icon) │  │   (icon) │  │   (icon) │
│  العنوان │  │  العنوان │  │  العنوان │
└──────────┘  └──────────┘  └──────────┘
```

**3. إبقاء رسالة الترحيب كما هي** — "مرحباً، اسم المستخدم" مع المعدل أو رسالة إكمال الملف.

### النتيجة
- جميع الخيارات مرئية في شاشة واحدة بدون تمرير كثير
- تجربة أسرع للوصول إلى أي قسم
- تصميم نظيف ومرتب يناسب الجوال
