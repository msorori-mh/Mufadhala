

## إصلاح رسالة صفحة الاختبار للمدير/المشرف

### المشكلة
عندما يكون المستخدم مديراً أو مشرفاً بدون تخصص، تظهر رسالة "اختر تخصصك أولاً من الملف الشخصي" وهي رسالة غير مناسبة لمستخدمي الإدارة.

### الحل
تعديل ملف واحد: `src/pages/ExamSimulator.tsx`

في الشرط الموجود بالسطر 462-478، التفريق بين حالتين:

1. **مستخدم إداري (isStaff)**: عرض رسالة مختلفة توضح أن محاكاة الاختبار مخصصة للطلاب، مع زر للعودة إلى لوحة التحكم.
2. **طالب بدون تخصص**: إبقاء الرسالة الحالية كما هي.

### التغيير
```tsx
if (!student?.major_id && !isOffline) {
  const isStaffNoMajor = isStaff;
  return (
    // ... header ...
    <main>
      {isStaffNoMajor ? (
        <>
          <GraduationCap icon />
          <p>"محاكاة الاختبار مخصصة للطلاب المسجلين"</p>
          <p>"كمسؤول، يمكنك متابعة أداء الطلاب من لوحة التحكم"</p>
          <Button → /admin/reports/exams>تقارير الاختبارات</Button>
        </>
      ) : (
        // الرسالة الحالية للطالب
      )}
    </main>
  );
}
```
