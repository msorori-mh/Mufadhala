

# خطة: دعم رفع ملفات متعددة في دليل التنسيق والتسجيل

## الوضع الحالي
حالياً كل جامعة تدعم رفع ملف واحد فقط (`guide_url`) كدليل تنسيق. المطلوب دعم رفع ملفات وصور متعددة.

## التغييرات المطلوبة

### 1. تحديث قاعدة البيانات
- إضافة عمود `guide_files` من نوع `jsonb` بقيمة افتراضية `'[]'` لجدول `universities`
- يحتوي مصفوفة من الكائنات: `[{ url, name, type, uploaded_at }]`
- الاحتفاظ بعمود `guide_url` الحالي للتوافق مع البيانات القديمة

### 2. تحديث صفحة إدارة الجامعات (`AdminUniversities.tsx`)
- تحويل منطقة الرفع من ملف واحد إلى رفع متعدد (`multiple` attribute)
- عرض قائمة الملفات المرفوعة مع إمكانية حذف كل ملف
- دعم أنواع: PDF, JPG, PNG, WebP
- عرض معاينة مصغرة للصور وأيقونة للملفات

### 3. تحديث دليل الكليات (`CollegeGuide.tsx`)
- عرض الملفات المتعددة في قسم "أدلة التنسيق والتسجيل"
- الصور تظهر كمعاينات قابلة للنقر (تفتح بحجم كامل)
- ملفات PDF تظهر كأزرار تحميل

### التفاصيل التقنية
- التخزين في bucket `university-guides` الموجود (عام)
- البنية في `guide_files`: `[{ url: string, name: string, type: "pdf"|"image", uploaded_at: string }]`
- نقل `guide_url` القديم تلقائياً إلى `guide_files` عند التعديل
