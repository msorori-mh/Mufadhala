

## إضافة شاشة انتقالية لتأكيد تفعيل الاشتراك

### الفكرة
عند دخول المستخدم صفحة الاشتراك واشتراكه فعّال (active وليس trial)، تظهر شاشة انتقالية جميلة لمدة 5 ثوانٍ تعرض رسالة التفعيل مع تفاصيل الخطة والمدة، ثم تختفي تلقائياً لتظهر العرض العادي.

### التغييرات — ملف واحد: `src/pages/Subscription.tsx`

**1. إضافة حالة `showActivationSplash`**
- `useState(true)` تبدأ بـ true عند وجود اشتراك active
- تُخفى تلقائياً بعد 5 ثوانٍ عبر `setTimeout`، أو يمكن للمستخدم الضغط على زر "متابعة"

**2. جلب اسم الخطة**
- استخدام `subscription.plan_id` لإيجاد الخطة من مصفوفة `plans` المحمّلة أصلاً
- عرض اسم الخطة وتاريخ الانتهاء

**3. تصميم الشاشة الانتقالية**
- شاشة كاملة بخلفية متدرجة خضراء
- أيقونة `CheckCircle` كبيرة مع حركة بسيطة (animate-bounce)
- نص "تم قبول طلب الدفع وتفعيل اشتراكك!"
- تفاصيل: اسم الخطة + تاريخ الانتهاء
- زر "متابعة" + عداد تنازلي يختفي بعد 5 ثوانٍ

**4. المنطق**
- تظهر فقط مرة واحدة لكل جلسة باستخدام `sessionStorage` (مفتاح `subscription_splash_shown`)
- إذا سبق عرضها في الجلسة الحالية، تُتخطّى مباشرة

```text
┌─────────────────────────────┐
│                             │
│        ✅ (كبيرة)           │
│                             │
│  تم قبول طلب الدفع         │
│  وتفعيل اشتراكك!           │
│                             │
│  الخطة: طبيب المستقبل      │
│  ينتهي في: 2026/07/09      │
│                             │
│      [ متابعة ]             │
│                             │
└─────────────────────────────┘
```

### النتيجة
تجربة مستخدم أفضل — يرى المستخدم تأكيداً واضحاً عند أول زيارة بعد تفعيل اشتراكه، ثم ينتقل للعرض العادي.
