

# إضافة نوعين جديدين لطرق الدفع: تحويل عبر الكريمي + نقطة حاسب

## المشكلة
أنواع طرق الدفع الحالية محدودة بـ 4 أنواع (bank, exchange, ewallet, network_transfer). المطلوب إضافة:
1. **تحويل عبر الكريمي** — يوضح اسم المستلم
2. **نقطة حاسب** — يوضح اسم الحساب ورقم الحساب

## التغييرات المطلوبة

### 1. تحديث قاعدة البيانات (Migration)
تعديل دالة التحقق `validate_payment_method_type` لإضافة النوعين الجديدين: `kuraimi_transfer` و `hasib_point`.

### 2. تحديث صفحة إدارة طرق الدفع (`AdminPaymentMethods.tsx`)
- إضافة خيارين جديدين في قائمة النوع: "تحويل عبر الكريمي" و "نقطة حاسب"
- إضافة أقسام عرض جديدة لكل نوع مع أيقونات مناسبة
- تحديث تسميات الحقول حسب النوع (مثل "اسم المستلم" للكريمي)
- إضافة فلترة للنوعين الجديدين

### 3. تحديث صفحة الاشتراك للطالب (`Subscription.tsx`)
- إضافة النوعين الجديدين في مصفوفة الأنواع المعروضة
- إضافة أيقونات وتسميات مناسبة لكل نوع

### ملاحظة تقنية
- الأيقونة المقترحة للكريمي: `Banknote` أو `Send`
- الأيقونة المقترحة لنقطة حاسب: `Monitor` أو `Laptop`
