

# إضافة الجدول الزمني للتنسيق وتعليمات التنسيق لبيانات الجامعة

## ملخص
إضافة حقلين جديدين لجدول الجامعات: **الجدول الزمني للتنسيق** (مراحل التنسيق بتواريخها) و**تعليمات التنسيق** (نص إرشادي). تظهر هذه البيانات تلقائياً في بطاقات الكليات التابعة لكل جامعة في صفحة دليل الكليات.

## التغييرات المطلوبة

### 1. تعديل قاعدة البيانات (Migration)
إضافة عمودين جديدين لجدول `universities`:
- `coordination_timeline` — من نوع `jsonb` — مصفوفة من الأحداث، كل حدث يحتوي على `phase` (اسم المرحلة) و`date` (التاريخ/الفترة)
- `coordination_instructions` — من نوع `text` — نص تعليمات التنسيق

### 2. تعديل صفحة إدارة الجامعات (`AdminUniversities.tsx`)
- إضافة قسم جديد في حوار التعديل/الإضافة يحتوي على:
  - **الجدول الزمني**: واجهة ديناميكية لإضافة/حذف مراحل التنسيق (اسم المرحلة + التاريخ)
  - **تعليمات التنسيق**: حقل نص طويل (Textarea)
- حفظ البيانات مع باقي بيانات الجامعة
- عرض أيقونة في قائمة الجامعات عند وجود جدول زمني

### 3. تعديل صفحة دليل الكليات (`CollegeGuide.tsx`)
- جلب بيانات `coordination_timeline` و`coordination_instructions` مع الجامعات (موجودة بالفعل في `select("*")`)
- عرض الجدول الزمني وتعليمات التنسيق تلقائياً في كل بطاقة كلية، مأخوذة من الجامعة التابعة لها
- إضافة قسم مميز بصرياً يعرض مراحل التنسيق كـ timeline بسيط
- عرض التعليمات كنص إرشادي أسفل كل بطاقة كلية

### 4. تحديث تصدير PDF
- إضافة عمود أو قسم في تصدير PDF يتضمن الجدول الزمني والتعليمات لكل جامعة

## التفاصيل التقنية

**هيكل `coordination_timeline` (jsonb)**:
```json
[
  { "phase": "المرحلة الأولى", "date": "1 - 15 سبتمبر 2025" },
  { "phase": "المرحلة الثانية", "date": "16 - 30 سبتمبر 2025" }
]
```

**الملفات المعدّلة**:
1. Migration جديد — إضافة العمودين
2. `src/pages/admin/AdminUniversities.tsx` — واجهة الإدخال
3. `src/pages/CollegeGuide.tsx` — عرض البيانات للطلاب
