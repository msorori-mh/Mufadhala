

# إضافة تقسيم المقررات حسب الصف الدراسي (أول / ثاني / ثالث ثانوي)

## الفكرة
كل مادة دراسية (كيمياء، أحياء، إلخ) تحتوي على مقررات من ثلاثة صفوف ثانوية. نضيف عمود `grade_level` للدروس لتصنيفها، ثم نعرضها مجمّعة حسب الصف في واجهة الطالب والإدارة.

## التعديلات

### 1. Migration: إضافة عمود `grade_level` لجدول `lessons`
```sql
ALTER TABLE public.lessons ADD COLUMN grade_level smallint DEFAULT NULL;
COMMENT ON COLUMN public.lessons.grade_level IS '1=أول ثانوي, 2=ثاني ثانوي, 3=ثالث ثانوي';
```
- القيمة اختيارية (NULL للدروس العامة التي لا تنتمي لصف معين)

### 2. تعديل `src/pages/admin/AdminContent.tsx`
- إضافة حقل اختيار "الصف الدراسي" في نموذج إنشاء/تعديل الدرس (قائمة منسدلة: أول ثانوي / ثاني ثانوي / ثالث ثانوي / بدون تصنيف)
- إضافة فلتر بالصف الدراسي في قائمة الدروس
- عرض badge الصف الدراسي على بطاقة كل درس

### 3. تعديل `src/pages/LessonsList.tsx` (واجهة الطالب)
- تجميع الدروس داخل كل مادة حسب الصف الدراسي
- عرض 3 بطاقات فرعية لكل مادة: "مقرر صف أول ثانوي"، "مقرر صف ثاني ثانوي"، "مقرر صف ثالث ثانوي"
- استخدام Accordion أو بطاقات قابلة للتوسيع لعرض الدروس تحت كل صف

### 4. تعديل `src/pages/LessonDetail.tsx`
- عرض الصف الدراسي كـ badge في تفاصيل الدرس
