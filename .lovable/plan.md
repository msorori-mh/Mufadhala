

## إضافة فلتر المواد لقسم "حالة إكمال الدروس" في صفحة تحليل الأداء

### الهدف
إضافة فلتر بالمادة الدراسية في قسم "حالة إكمال الدروس" بصفحة `StudentPerformance.tsx`، بحيث يمكن للطالب عرض حالة إكمال الدروس لكل مادة على حدة مع عرض إحصائيات الإكمال لكل مادة.

### التغييرات في `src/pages/StudentPerformance.tsx`

**1. جلب بيانات المواد (subjects)**
- تعديل الاستعلام الحالي لجلب `subject_id` مع الدروس (موجود فعلاً في السطر 68)
- جلب أسماء المواد من جدول `subjects` بناءً على `subjectIds` المتاحة

**2. إضافة state لفلتر المادة**
- إضافة `useState` لتخزين المادة المختارة (`selectedSubjectId: string | "all"`)

**3. إضافة شريط فلتر أفقي (chips/tabs)**
- عرض أزرار أفقية قابلة للتمرير أعلى قائمة الدروس
- زر "الكل" + زر لكل مادة مع عدد الدروس المكتملة/الإجمالي
- تصميم compact يناسب الموبايل

**4. تصفية قائمة الدروس**
- عند اختيار مادة، تُعرض فقط الدروس التابعة لها
- تحديث عداد الإكمال ليعكس المادة المختارة
- عرض نسبة إكمال المادة المختارة في شريط Progress

### التفاصيل التقنية
- ربط كل درس بمادته عبر `subject_id` الموجود في استعلام الدروس
- جلب `subjects` (name_ar) عبر استعلام إضافي ضمن `Promise.all` الحالي
- الفلتر يعمل بالكامل على البيانات المحملة مسبقاً (client-side filtering)
- لا حاجة لتعديل قاعدة البيانات أو إضافة جداول جديدة
