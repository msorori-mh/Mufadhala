## 📜 نظام الترقيم

نتبع [Semantic Versioning](https://semver.org/lang/ar/) بصيغة `MAJOR.MINOR.PATCH`:

- **MAJOR** — تغييرات جذرية غير متوافقة (تحديث قاعدة بيانات كاملة، تغيير معمارية)
- **MINOR** — ميزات جديدة متوافقة مع السابق
- **PATCH** — إصلاح أخطاء فقط

### حساب `versionCode`:
```
versionCode = MAJOR × 10000 + MINOR × 100 + PATCH
مثال: 5.1.0 → 5×10000 + 1×100 + 0 = 50100
```

## 📋 سجل الإصدارات

### 🆕 v5.1.0-stable (50100) — 2026-04-16

#### ✨ ميزات جديدة
- **إدارة النماذج**: إضافة عداد الأسئلة لكل نموذج في صفحة الإدارة
- **فلاتر متقدمة**: فلترة النماذج حسب السنة/الجامعة/الحالة (منشور/مسودة)
- **نسخ النموذج**: زر سريع لنسخ نموذج كامل (مع كل الأسئلة) بنقرة واحدة
- **إلغاء النشر السريع**: زر بجانب النماذج المنشورة لإرجاعها مسودة فوراً

#### 📜 تحديثات سياسية وقانونية
- **سياسة الخصوصية**: إعادة كتابة كاملة لتطابق نموذج Google Play Data Safety
- **نموذج Data Safety**: توليد PDF رسمي + JSON/CSV احتياطي للمراجعات المستقبلية
- **معالجو البيانات**: توضيح صريح لـ Supabase + Google Gemini كأطراف ثالثة
- **مدد الاحتفاظ**: تحديد مدد دقيقة لكل نوع بيانات (OTP: 5 د، إيصالات: 12 ش)

#### 🛠️ تحسينات تقنية
- إضافة سكريبت `build-release.sh` لأتمتة البناء بأمر واحد
- التحقق التلقائي من نظافة `capacitor.config.ts` قبل البناء
